# InkCloud

Aplicación web de gestión de notas estilo Google Keep, desarrollada como proyecto del Grado Superior de **Desarrollo de Aplicaciones Web (DAW)**.

![Logo](Img/Logo_InkCloud.png)

---

## Descripción

InkCloud permite crear, organizar y gestionar notas de forma visual e intuitiva. Las notas se almacenan en **LocalStorage**, por lo que persisten entre sesiones sin necesidad de backend.

El proyecto abarca dos asignaturas: **Diseño de Interfaces** y **Desarrollo Web en Entorno Cliente**.

---

## Funcionalidades

- **Crear notas** con título, texto, prioridad, color de fondo y color de texto
- **Sistema de prioridad**: las notas prioritarias aparecen en la parte superior
- **Buscar notas** por título, prioridad o color
- **Editar y eliminar** notas individualmente
- **Fondos personalizados**: color sólido, degradado lineal/radial/cónico o imagen de fondo
- **Estadísticas**: número de notas activas, eliminadas, modificadas, letras totales y media
- **Barra de navegación fija** que sigue el scroll
- **Barra de redes sociales** (X, Instagram, Snapchat) flotante
- **Widget de chatbot** en la esquina inferior derecha con GIF animado
- **Persistencia** completa mediante LocalStorage
